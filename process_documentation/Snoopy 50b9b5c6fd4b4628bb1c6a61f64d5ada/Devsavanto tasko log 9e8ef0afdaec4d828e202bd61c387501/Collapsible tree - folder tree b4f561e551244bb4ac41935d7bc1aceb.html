El formato es aparentemente un json de carpetas y archivos dentro de las carpetas. A partir de un tree de linux lo podemos construir.

El árbol del proyecto lo podemos generar con esto:

tree <folder-name> -J --gitignore -o <filename>.json

El output de jerarquía debería verse como esto:

{
  "name": "Eve",
  "children": [
    {
      "name": "Cain"
